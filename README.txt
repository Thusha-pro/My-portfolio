BEFORE YOU DEPLOY: ADD YOUR CV
----------------------------------------
The site has a "Download CV" button that looks for a file named:

    Kannan_Thushanthan_CV.pdf

Copy your CV PDF into THIS folder and rename it to exactly that
(spelling and capitalisation matter). The button will work
automatically once the file is there.

If you skip this step the site still works — the CV button just
won't download anything.


----------------------------------------
DEPLOY TO NETLIFY (FREE, ~30 SECONDS)
----------------------------------------
1. Go to https://app.netlify.com/drop
   (or sign up free at netlify.com first if you want a dashboard)

2. Drag this WHOLE FOLDER (not just index.html) onto the upload area.

3. Netlify gives you a live URL instantly, e.g.
   https://something-random.netlify.app

4. To customise the URL:
   - Click "Site settings" -> "Change site name"
   - Pick something like:  thushanthan-kannan
   - Your URL becomes:     thushanthan-kannan.netlify.app

5. To use a custom domain (e.g. yourname.com):
   - Buy a domain (Namecheap, Porkbun, etc. ~£8-12/year)
   - Site settings -> Domain management -> Add custom domain
   - Follow Netlify's DNS instructions


----------------------------------------
UPDATING YOUR SITE LATER
----------------------------------------
Whenever you want to change something (add a new publication,
update your CV, change wording):

1. Open index.html in any text editor (VS Code, Notepad, TextEdit).
2. Find the section you want to edit and change the text.
3. Save the file.
4. Go to your Netlify site dashboard -> "Deploys" -> drag the
   updated folder onto the page. Live in seconds.

Or just ask Claude again — paste in what you want changed and
I'll give you an updated file.
